# Finnovative_Project_AA: Z-Shield Security Center

Đây là mã nguồn cho dự án Z-Shield - Hệ thống giả lập trung tâm điều hành bảo mật (SOC), được xây dựng bằng Python và Streamlit.

## Tính năng
- **Z-Face:** Xác thực khuôn mặt bảo mật cao (Mô phỏng Zero-Knowledge Proof).
- **Z-Geo:** Làm nhiễu vị trí GPS.
- **Z-Age:** Xác thực độ tuổi 18+.
- **Đa ngôn ngữ:** Hỗ trợ Tiếng Việt và Tiếng Anh.

## Hướng dẫn triển khai (Deployment Instructions)
Để chạy dự án này trên máy cục bộ, vui lòng làm theo các bước sau:

### 1. Yêu cầu hệ thống
- Python 3.8 trở lên được cài đặt trên máy.

### 2. Cài đặt thư viện
Mở terminal (hoặc Command Prompt) tại thư mục chứa code và chạy lệnh:
                pip install streamlit pandas numpy
### 3. Chạy ứng dụng trên Terminal
Sử dụng lệnh sau để khởi chạy ứng dụng:
streamlit run FINAL.py
